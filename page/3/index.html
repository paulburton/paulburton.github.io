Yup, more Axdroid Aug 10, 2010 New build Axdroid 2010-08-09 here! Changes since Saturdays build: JIT is enabled. It turns out the fancy new JIT compiler Android 2.2 introduced was disabled for ARMv5TE devices for some reason. This build has it enabled (actually I’m now using the dalvik code from the CyanogenMod ROMs). CompCache is included. This takes a chunk of RAM and uses it as a compressed swap device, so essentially you can fit more in there at the expense of a little processor time doing the (de)compression. ... ➦
